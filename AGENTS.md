# EarthReadyMXFinal Agent Guide

This repository is an iOS application packaged through Swift Package Manager.

## Commands

- Build: `swift build`
- Test: `swift test` (if test targets exist)

## Stack notes

- Platform target: iOS 16
- The package is centered on the `AppModule` executable target
- `Package.swift` is autogenerated and explicitly warns not to edit it by hand

## Rules

- Do not hand-edit autogenerated package metadata unless the task explicitly
  requires regenerating it through the proper tool.
- Keep iOS-specific behavior explicit and narrowly scoped.
- If the repository has no tests, report that clearly instead of pretending
  validation exists.
